Todo Card Component

A responsive and interactive **single Todo Task Card** built with vanilla JavaScript.
This project focuses on DOM manipulation, state management, and dynamic UI updates without any frameworks.

A couple of changes were made after the first task, I added

**Task Status Management**
  Switch between *Pending*, *In Progress*, and *Done*

**Live Deadline Timer**
  Displays time remaining and updates automatically

**Overdue Detection**
  Highlights tasks when the deadline has passed

**Edit Mode**
  Update title, description, priority, and due date

**Expandable Description**
  Toggle between preview and full task details

**Completion Toggle**
  Mark tasks as completed with visual feedback
